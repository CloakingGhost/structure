완전 이진 트리

최대 힙(max heap)
부모 노드의 값 >= 자식 노드의 값
루트 노드 == 트리의 최대값

최소 힙(min heap)
부모 노드의 값 <= 자식 노드의 값
루트 노드 == 트리의 최소값

값 중복 허용
좌우 값의 제약은 없다
상하 관계에서 값의 제약이 있음

최대/최소 연산이 빠르다 O(1)
삽입: O(logN)
삭제: O(logN)

우선순위 큐에 사용됨
운영체제의 작업 스케줄링

힙 정렬에도 사용됨

=================
Heapify(힙을 재구조화 하는 연산)

힙의 재구조화
데이터 추가/삭제 후에도 속성을 유지하도록 함

Min Heap 데이터 삽입
1. 완전 이진 트리의 형태를 유지해야 하므로 우선 leaf 노드에 삽입
2. 힙의 조건을 만족하는지 확인
3. 삽입된 노드와 부모 노드의 값을 바꿈

Max Heap에서 최대값 삭제
1. 마지막 노드를 루트 노드로 가져옴
2. 힙의 조건을 만족하는지 확인
3. 자식 노드와 위치를 바꿈
